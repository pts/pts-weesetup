pts-weesetup: self-contained source code of weesetup.exe
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^
weesetup.exe is a Windows command-line tool which can write a boot manager
(typically the Wee boot manager, wee63.bin) to the MBR (Master Boot Record) of
a physical device (drive) connected to Windows. pts-weesetup is a
self-contained source code repository which can be cross-compiled on Linux
to weesetup.exe using GCC and MinGW (i686-w64-mingw32-gcc).

Downloaded https://storage.googleapis.com/google-code-archive-source/v2/code.google.com/grubutils/source-archive.zip
on 2018-01-04, contains the latest weesetup sources as of 2016-02-15.
